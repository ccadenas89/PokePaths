# PokePaths
Una pantalla de entrada inspirada en un menú de Pokémon roguelike.

## Archivos
- `index.html` — pantalla principal.
- `styles.css` — estilos de interfaz inspirados en el ejemplo.
- `script.js` — interacciones simples de botones.

## Uso
Abre `index.html` en un navegador para ver la pantalla principal.

> Por ahora no se han añadido las opciones de Hall of Fame ni la conexión con Discord.
